# impulse-gui
*Impulse Personal Health Manager* (Impulse PHM) is a desktop application that 
empowers users to own and manage their personal health information. 
Specifically, `impulse-gui` is the graphical user interface (GUI) that users 
interact with. Additionally, it calls into the core library to do the 
heavy-duty work (via C++ and Rust integration).

## Status
The project is actively maintained, but it's not ready for production as many 
core features are still being implemented and tested.
